## 3. 養龍蝦不能錯過的無頭瀏覽器 - pinchtab
- **URL:** https://www.threads.com/@min.biz/post/DVFgHwbCeZv
- **作者:** min.biz
- **翻譯:**
養龍蝦不能錯過的無頭瀏覽器！！！
github.com/pinch…

- 獨立 http server，任何 agent / 語言 / curl 都能用
- 超輕量，12MB Go binary
- 每頁約 800 tokens，相比 OpenClaw 每頁大約 10,000 tokens，爬網頁更省
- A11y snapshots + element interaction
- 支援無痕模式，可以繞過不少防 bot 網站
- Session persistence — 登入一次，重啟後仍保持登入

目前 OpenClaw 的 Browser Relay 太不穩定，常常會自己斷開連線，要一直遠端連回龍蝦🦞手動重啟

在測試用它幫我爬 Threads 文章，效果還不錯，尤其持續化這件事情，非常方便，登入一次就能用
安裝過程也很無腦，我直接丟給龍蝦這個 Github Repo，它直接幫我做完所有安裝🤣

- **摘要:** 推薦 pinchtab 無頭瀏覽器，強調省 token、持續化登入等優點，可作為 OpenClaw Browser Relay 的替代方案。
